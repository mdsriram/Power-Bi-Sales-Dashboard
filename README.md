Overview
This project presents a comprehensive sales analysis dashboard built using Power BI. It provides insights into sales trends, Profit relationships, promotions and Geographic Distribution.

Key Features
1. Profit vs Net Sales relationship analysis
2. City wise sales distribution using map visualization
3. Promotion based discount analysis 
4. Sales trends over time 
5. KPI Card showing total number of orders

Tools Used
1. Power BI
2. Excel

Key Insights
1. Strong relationship between prfit and net sales
2. Weekend Flash sales provided the highest discounts
3. Sales show consistn=ent fluctuations with periodic peaks
4. Certain cities contribute significantly more to sales

1. Sales Project.pbix
2. Sales.xlsx
